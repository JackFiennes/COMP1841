Entity names look like this: &entity_name; and entity numbers look like this: &entity_number.

To display a less than sign we must write < or <.

A commonly used HTML entity is the non - breaking space: " "

Two words seperated by a non - breaking space will stick together and not break into a new line.

The non - breaking hyphen ‑ is used to define a hyphen character that does not break into a new line.

The next topic to cover it UTF-8 encoding: What is it?

A useful explainer blog.

It stands for Unicode Transformation Format - 8 bits.

Computers use a binary system to store data in a sequence of 1's and 0s.

The most basic unit of binary is a bit, which is a single 1 or 0 and the next largest unit of binary, a byte, consists of 8 bits.

A kilobyte is roughly one thousand bites and a gigabyte is roughly one billion bytes.

UTF-8 is unique because it can transform all Unicode characters, unicode being the system which assigns a unique code to each character, to a matching unique binary string and vice versa.

UTF-16 is a variable-length character encoding used in all major operating systems like Windows, IOS, and Unix.
